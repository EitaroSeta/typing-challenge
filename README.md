# タイピングチャレンジ

ブラウザで動作するタイピングバトルゲーム。10波の敵を倒しながら最終ボス「ハマーン」を目指せ！

## 遊び方

`index.html` をブラウザで開くだけで起動します。インストール・ビルド不要。

1. 言語モードを選択（英語 / 日本語ローマ字 / MIX）
2. 画面に表示された単語をキーボードで入力
3. 制限時間内に打ち切ると敵にダメージ
4. 10波クリアで VICTORY！

## 機能

- 英語・日本語（ローマ字入力）・MIX の3モード
- Web Audio API による効果音・BGM（すべて手続き的生成）
- コンボボーナス・タイムボーナスによるスコア計算
- ミュートボタン対応

## 技術スタック

- バニラ HTML / CSS / JavaScript（単一ファイル）
- 外部依存なし
